Kinetic Wave

Kinetic Wave is a synthesizer/modulator that uses a kinect to track body movements and modulate audio samples. This project was made in C++ using the JUCE framework and the Kinect SDK, with a small self-made interface using NuiApi to track specific body parts.

Description

KineticWave was initially ment to be a NIME (New Interface for Musical Expression), but I quickly realized it could also be used as a tool to create soundscapes.

KineticWave can modulated 4 different effects on 2 different audio samples; audio, playrate/pitch, distortion and phase modulation. It also offers the possibility to use tresholds instead of continious modulation or just disabled certain effects intearly. The video shows the different modulations and its effect based on the hand positions.

The soundscape example shows the use of KineticWave to add texture in a more controlled 'analog' way. This soundscape was not entierly made with KineticWave, but some of the textures were created using it.

Skeleton Tracking
.h .cpp

This is a very simple interface to track specific joints using NuiApi to access the Kinect sensor's data. Joints can be registered for tracking to follow thier position. To update the internal data of the joints, void Update() has to be manually called, this prevents unnecessary data copying.

Phase Modulation
.h .cpp

This phase modulation effect consists of having multiple delayed voices with each there own modulating volume. By using std::queue, a delay can be implemented very easily. Each voice has its own queue where only the back will be read and used as sample. If you push every sample that has to be delayed, and read the back of the queue as the delayed before popping. Then this means that the delay time can be controlled by changing the size of the queue.
