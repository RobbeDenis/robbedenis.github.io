Kinetic Wave

Kinetic Wave is a synthesizer/modulator that uses a kinect to track body movements and modulate audio samples. This project was made in C++ using the JUCE framework and the Kinect SDK, with a small self-made interface using NuiApi to track specific body parts.

Description

KineticWave was initially meant to be a New Interface for Musical Expression or NIME in short, but it turned out to be more effective as a sound design tool to create interesting and experimental soundscapes.

Kinetic Wave can modulate 4 different effects on 2 different audio samples; audio, playrate/pitch, distortion and phase modulation. It also offers the possibility to use thresholds instead of continuous change or just disable certain effects intearly.

Skeleton Tracking
.h .cpp

This is a very simple interface to track specific joints using NuiApi to access the Kinect sensor's data. Joints can be registered for tracking to follow their position. To update the internal data of the joints, update has to be manually called, this prevents unnecessary data copying.
