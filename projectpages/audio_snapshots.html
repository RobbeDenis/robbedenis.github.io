Metasound Audio Snapshots

Metasound Audio Snapshots is a plugin for Unreal Engine 5 that makes it possible to record and playback specific parts of a Metasound graph at runtime. This plugin was made to record and use audio data outside of the Metasound system and reduce the CPU usage by giving control over what and when audio generation can be replaced by its own 'cached' version.

Description

Metasound is Unreal Engine's system for realtime audio synthesis and processing. It provides a node based interface for creating complex audio with sample accurate control. This plugin extends this system by allowing users to use cached versions of the audio as playback or data outside of Metasound.

The snapshot node can be triggered to start recording the audio for a specified duration or size. On completion, a signal is sent out and the snapshot is ready to be used. (look at the project again for other triggers)
The snapshot is also capable of resizing the buffer to specified frequencies without changing the effective buffer size, this way the snapshots can be used to loop repetative signals with minimal artifacts. (maybe about other sizing modifiers and downsampling?)
